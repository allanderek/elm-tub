Introduction

This is just a collection of simple little Elm examples/projects.

Noughts and Crosses

The React tutorial gives a version of tic-tac-toe (which folk in the UK, including me, call noughts and crosses), so I've done roughly the same thing but in Elm. It has a move history as well as the usual rules for noughts and crosses.

A Text Editor in Elm

A simple text editor in Elm. Click in the main div to focus the editor div, and then normal text editing functions work. No selection or selection highlighting as yet. The cursor also changes width a little for a reason I'm not quite sure, but believe that it is fixable in CSS.
